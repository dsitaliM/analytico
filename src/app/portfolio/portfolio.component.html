Agri-Analytico was founded in August 2018 as part of the ZICTA Innovation Programme. Here are some highlights of the progress made so far.

Working Prototype

Our device currently measures soil moisture, humidity, light intensity, and temperature. It sends the data in real time to the cloud for analysis. The device can be easily extended to provide remote control of irrigation systems.

Mobile Application

Using the mobile application growers can monitor the data from the sensor device. They can view the field parameters in real time and make informed decisions.


Future Design

This is our envisioned design of the sensor device. It will be small, solar powered, and extensible. It will be Zero Maintenacne too, users will not have to change batteries or service the device.
